## Context

PLAN-003 needs a reliable automated test that exercises the full nexusOrchestrator pipeline: task submission → LLM generation → file output → status completion. This test uses a real SQLite DB (temp file), a mock LLM that returns a known string, and the actual HTTP API. It validates that the daemon works end-to-end without needing a real LLM provider.

This test is the primary acceptance gate before dogfooding with a real LLM.

### Test 1: TestFullPipelineHTTP

1. Create temp SQLite DB
2. Create mock LLM (implements `ports.LLMClient`):
   - `Ping() bool` → true
   - `ProviderName() string` → "MockLLM"
   - `GenerateCode(prompt string) (string, error)` → "// generated by MockLLM\nfunc Foo() {}"
   - `Chat(messages []domain.Message) (string, error)` → calls GenerateCode on last message
   - `GetAvailableModels() ([]string, error)` → ["mock-model"]
3. Create temp dir for file output
4. Wire: `repo → discoverySvc(mockLLM) → orchestrator → httptest.NewServer(httpapi.StartServer)`
5. POST to `/api/tasks`:
   ```json
   {"projectPath": "<tmpdir>", "targetFile": "out.go", "instruction": "generate a foo function"}
   ```
6. Poll `GET /api/tasks/{id}` until status == "COMPLETED" (timeout: 10s)
7. Assert file `<tmpdir>/out.go` contains "generated by MockLLM"
8. Assert `GET /api/tasks/{id}` → `{"status":"COMPLETED"}`

### Mock LLM Implementation:

```go
type mockLLM struct{}

func (m *mockLLM) Ping() bool                                    { return true }
func (m *mockLLM) ProviderName() string                         { return "MockLLM" }
func (m *mockLLM) GetAvailableModels() ([]string, error)        { return []string{"mock-model"}, nil }
func (m *mockLLM) GenerateCode(prompt string) (string, error)   { return "// generated by MockLLM\npackage main", nil }
func (m *mockLLM) Chat(msgs []domain.Message) (string, error)   { return m.GenerateCode("") }
```

### HTTP wiring for test (simpler than StartServer — use in-process):

```go
func newTestServer(t *testing.T) (orch ports.Orchestrator, srv *httptest.Server, cleanup func()) {
    dbFile, _ := os.CreateTemp("", "nexus-test-*.db")
    dbPath := dbFile.Name()
    dbFile.Close()

    repo, err := repo_sqlite.New(dbPath)
    require.NoError(t, err)

    tmpDir := t.TempDir()
    writer := fs_writer.New()
    
    discovery := services.NewDiscoveryService(&mockLLM{})
    sessionRepo := repo_sqlite.NewSessionRepo(repo)
    orch = services.NewOrchestrator(discovery, repo, writer, sessionRepo)

    // Wire HTTP server using httptest
    srv = httptest.NewServer(/* httpapi handler */)

    cleanup = func() {
        orch.(*services.OrchestratorService).Stop()
        repo.Close()
        srv.Close()
        os.Remove(dbPath)
    }
    return
}
```

- [ ] Mock LLM implements `ports.LLMClient` interface completely

## Anti-patterns to Avoid

- NEVER use real LLM providers in integration tests — use mock
- NEVER use fixed port numbers (risk of port conflict) — use `httptest.NewServer` (port 0)
- NEVER use `time.Sleep(N)` for polling — use polling with deadline
- NEVER delete DB files with `os.Remove` before `repo.Close()` — always close before delete
