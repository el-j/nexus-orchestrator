🔧 Build from Source

Requires Go 1.24+ and CGO enabled. CGO is needed for the SQLite session store.

Headless Daemon

git clone https://github.com/el-j/nexusOrchestrator
cd nexusOrchestrator
CGO_ENABLED=1 go build \
  -o nexus-daemon \
  ./cmd/nexus-daemon/...

./nexus-daemon

Desktop App (requires Wails)

# Install Wails
go install github.com/wailsapp/wails/v2/cmd/wails@latest

# Build desktop app
wails build

# Or run in dev mode
wails dev

⚡ Quick Install (Linux / macOS)

Download and install the latest daemon binary with a single command:

Terminal 
curl -sSL https://github.com/el-j/nexusOrchestrator/releases/latest/download/nexus-daemon-$(uname -s | tr '[:upper:]' '[:lower:]')-$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/') -o nexus-daemon && chmod +x nexus-daemon
ℹ️

Note: Always verify the SHA256 checksum after downloading to ensure the binary has not been tampered with.

🔐 Verify Checksums

Verify the integrity of downloaded binaries using SHA256 checksums published with each release.

macOS / Linux

# Download checksum file
curl -sSL https://github.com/el-j/nexusOrchestrator/releases/latest/download/checksums.txt -o checksums.txt

# Verify (macOS)
shasum -a 256 -c checksums.txt

# Verify (Linux)
sha256sum -c checksums.txt

Windows (PowerShell)

# Download and verify
$file = "nexus-daemon-windows-amd64.exe"
$hash = Get-FileHash $file -Algorithm SHA256
Write-Host $hash.Hash

# Compare with published checksum at:
# https://github.com/el-j/nexusOrchestrator/releases/latest

📋 System Requirements

🖥️

Operating System

macOS 12+, Windows 10/11, Ubuntu 20.04+ or any modern Linux distro

💾

Memory

64 MB RAM minimum for the daemon. More required if running LLMs locally (typically 4–32 GB).

💿

Disk Space

~20 MB for the binary. SQLite session databases grow based on task history volume.

🌐

Network

Local network access to LLM backends. Internet required only for cloud providers (OpenAI, Anthropic).
