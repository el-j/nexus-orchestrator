API Reference

Complete HTTP REST API and MCP endpoint documentation. Base URL: http://localhost:9999  ·  MCP: http://localhost:9998/mcp

HTTP REST API

All endpoints accept and return application/json unless stated otherwise. The API runs on port 9999 by default.

ℹ️

Base URL: http://localhost:9999  —  Override the port with the NEXUS_HTTP_PORT environment variable.

Tasks

POST /api/tasks Submit a new code-generation task

Request Body

{
  "projectPath": "/path/to/project",
  "targetFile":  "output.go",
  "instruction": "Write a function that sorts strings",
  "contextFiles": ["main.go", "utils.go"],
  "modelId":      "codellama",
  "providerHint": "LM Studio",
  "command":      "execute"
}
FieldRequiredDescription
projectPathYesAbsolute path to project directory
targetFileYesRelative path for the generated output file
instructionYesNatural language prompt for the LLM
contextFilesNoFiles to include as context
modelIdNoConstrain to a specific model
providerHintNoPrefer a specific provider
commandNoplan, execute, or auto (default)

Response — 201 Created

{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "projectPath": "/path/to/project",
  "targetFile": "output.go",
  "instruction": "Write a function that sorts strings",
  "status": "QUEUED",
  "command": "execute",
  "createdAt": "2025-01-01T00:00:00Z",
  "updatedAt": "2025-01-01T00:00:00Z"
}
GET /api/tasks List all pending tasks (QUEUED or PROCESSING)

Query Parameters

None — returns all QUEUED and PROCESSING tasks.

Response — 200 OK

[
  {
    "id": "...",
    "status": "QUEUED",
    "instruction": "...",
    "projectPath": "...",
    "createdAt": "2025-01-01T00:00:00Z"
  }
]
GET /api/tasks/{id} Retrieve a single task by ID

Path Parameters

id — the UUID of the task

Response — 200 OK / 404

{
  "id": "a1b2c3d4-...",
  "status": "COMPLETED",
  "logs": "func sortStrings(s []string) []string {\n  sort.Strings(s)\n  return s\n}",
  "updatedAt": "2025-01-01T00:00:04Z"
}
DELETE /api/tasks/{id} Cancel a queued task

Path Parameters

id — the UUID of the task to cancel. Task must be in QUEUED state.

Response

204 No Content on success

404 Not Found if task doesn't exist or is already in a terminal state

GET /api/tasks/{id}/stream Stream task status updates via Server-Sent Events SSE

Headers

Accept: text/event-stream
Cache-Control: no-cache

Event Stream

data: {"status":"PROCESSING","updatedAt":"..."}

data: {"status":"COMPLETED","logs":"...","updatedAt":"..."}

The stream closes automatically when the task reaches a terminal state.

Providers

GET /api/providers List all registered providers with health status

No Parameters

Returns all registered providers and their current liveness status.

Response — 200 OK

[
  {
    "name": "LM Studio",
    "baseUrl": "http://127.0.0.1:1234",
    "active": true,
    "activeModel": "codellama:latest",
    "models": ["codellama:latest", "deepseek-coder:6.7b"]
  },
  {
    "name": "Ollama",
    "baseUrl": "http://127.0.0.1:11434",
    "active": false
  }
]
POST /api/providers Register a new LLM provider

Request Body

{
  "name":    "My Custom LLM",
  "baseUrl": "http://192.168.1.100:8080",
  "apiKey":  "optional-key",
  "models":  ["llama3", "mistral"]
}

Response — 201 Created

{
  "name":    "My Custom LLM",
  "baseUrl": "http://192.168.1.100:8080",
  "active":  false
}
DELETE /api/providers/{name} Unregister a provider

Path Parameters

name — URL-encoded provider name (e.g. LM%20Studio)

Response

204 No Content on success

Health & Utilities

GET /health Daemon health check

No Parameters

Response — 200 OK

{"status": "ok"}
GET /api/session/{projectPath} Get conversation history for a project

Path Parameters

projectPath — base64-encoded absolute project path

Response — 200 OK

[
  {"role": "user",      "content": "Write a sort function"},
  {"role": "assistant", "content": "func Sort(...)..."}
]

MCP Server API

The MCP server runs on port 9998 and implements JSON-RPC 2.0. It exposes six tools compatible with Claude Desktop and any MCP client.

ℹ️

MCP Endpoint: http://localhost:9998/mcp — See the MCP Integration page for Claude Desktop configuration.

Tool Description Required Args
submit_task Submit a code-generation task with project path, target file, and instruction projectPath, targetFile, instruction
get_task Retrieve the status and output of a task by its ID id
get_queue List all pending (QUEUED/PROCESSING) tasks in the queue —
cancel_task Cancel a queued task before it is processed id
get_providers List all registered LLM providers and their health status —
health Check if the orchestrator daemon is running and responsive —

Example: submit_task

Request JSON-RPC 2.0 
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "tools/call",
  "params": {
    "name": "submit_task",
    "arguments": {
      "projectPath": "/path/to/project",
      "targetFile":  "handler.go",
      "instruction": "Add error handling to the HTTP handler",
      "command":     "execute"
    }
  }
}
Response JSON-RPC 2.0 
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "{\"id\":\"abc-123\",\"status\":\"QUEUED\"}"
      }
    ]
  }
}

Example: get_task

Request JSON-RPC 2.0 
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "tools/call",
  "params": {
    "name": "get_task",
    "arguments": {
      "id": "abc-123"
    }
  }
}
Response JSON-RPC 2.0 
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "{\"id\":\"abc-123\",\"status\":\"COMPLETED\",\"logs\":\"...\"}"
      }
    ]
  }
}

Environment Variables

Variable Default Description
NEXUS_HTTP_PORT 9999 HTTP API and dashboard port
NEXUS_MCP_PORT 9998 MCP JSON-RPC server port
NEXUS_DB_PATH ~/.nexus/sessions.db SQLite database path
NEXUS_LOG_LEVEL info Log verbosity: debug, info, warn, error
OPENAI_API_KEY — Enables OpenAI provider
ANTHROPIC_API_KEY — Enables Anthropic provider
GITHUB_TOKEN — Enables GitHub Copilot provider
NEXUS_CONTEXT_LIMIT 128000 Default token limit for context guard
